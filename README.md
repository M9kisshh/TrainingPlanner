## Описание программы
Training Planner — это GUI-приложение для планирования и учёта тренировок. Позволяет добавлять записи с датой, типом и длительностью, просматривать их в таблице, фильтровать по типу и дате. Все данные сохраняются в JSON-файл и не требуют установки дополнительных библиотек.

---

## Требования для запуска
- Python 3.10 или выше  
- Никаких внешних библиотек – используется только стандартная библиотека tkinter, json, datetime.

## Как запустить
```bash
git clone https://github.com/M9kisshh/training-planner.git
cd training-planner
python main.py
